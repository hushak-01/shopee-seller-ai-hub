# 🛒 Shopee Seller AI Hub · 跨境卖家智能运营助手

> 产品经理作品集 · 求职项目 · 2026年3月

**[🛒 点击查看交互Demo]([https://github.com/hushak-01/shopee-seller-ai-hub](https://hushak-01.github.io/shopee-seller-ai-hub/))**



本项目为面向Shopee跨境卖家的官方原生AI运营工具产品方案，包含完整PRD文档与高保真交互原型。

**背景来源：** 在研究Shopee卖家生态时，我访谈了3位有东南亚跨境经验的卖家，发现他们共同面临三个核心痛点：选品依赖感觉而非数据、文案本地化耗时且违规率高、合规规则变化快但预警滞后。结合Shopee 2026年全托管战略方向，我选择设计这款内嵌于卖家中心的AI工具，而非第三方工具，核心逻辑是：**官方数据打通 + 零额外成本 + 合规实时同步**，这是任何第三方工具无法复制的壁垒。

---

## 产品核心功能

| 模块 | 解决的核心问题 | 优先级 |
|------|--------------|--------|
| 🔍 AI选品中心 | 卖家选品无数据支撑，蓝海品类找不到 | P0 |
| ✨ 本地化优化 | 翻译不准、标题无SEO、违规词过滤缺失 | P0 |
| 🛡️ 合规中心 | 合规规则多变，预警滞后，整改无引导 | P0 |
| 📊 数据看板 | 数据零散，无优化建议，复盘无方向 | P0 |
| 💰 智能定价 | 跨境成本核算复杂，定价失误导致亏本 | P1 |

## Demo截图预览
